## How to use

> **NOTE:** You need to have [Git](https://git-scm.com) installed on your machine.

```bash
# Clone this repository
$ git clone <repo_url>
# Go into the repository
$ cd downloader-api
# Copy env.example to .env
$ cp .env.example .env
# Install dependencies
$ npx yarn install
# Start the server
$ npx yarn start
```
